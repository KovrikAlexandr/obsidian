## Определение
**SSH-агент** – это фоновый процесс, который может сохранить ключ, позволяя вводить пароль к ключу (passphrase) только при сохранении. Далее ключ используется свободно.

Агент сохраняет расшифрованный после введения пароля приватный ключ. Любой процесс может запросить этот ключ у агента и агент его отдаст, поэтому использование `ForwardAgent` в конфиге может быть рискованным.
## Запуск агента
Запустить можно так:
```bash
eval `ssh-agent`
```

Команда запустит процесс и установит следующие переменные:
- `SSH_AUTH_SOCK` – сокет, по которому утилита ssh обращается к агенту
- `SSH_AGENT_PID` – PID запущенного агента

Убить процесс можно так:
```bash
eval `ssh-agent -k`
```

## Добавление ключей
Добавить ключ можно следующим образом:
```bash
ssh-add ~/.ssh/id_rsa
```

Посмотреть все ключи можно так:
```bash
ssh-add -l
```

Удалить ключ из агента:
```bash
ssh-add -d ~/.ssh/id_rsa
```

Удалить все ключи:
```bash
ssh-add -D
```
