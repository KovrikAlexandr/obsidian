Модуль ядра – объект, содержащий код, который расширяет функционал ядра. Сам модуль это `*.ko` файл.

В отличии от обычной программы:
- Запускается в kernel mode
- Программа запустится через `execve()` и получит свое адресное пространство. Модуль же запустится в адресном пространстве ядра

Список модулей:
```bash
lsmod
```

Информация о модуле:
```bash
modinfo <module name>
```

Удалить модуль (deprecated):
```bash
rmmod <module name>
```

Добавить модуль (deprecated):
```bash
insmod <module name>
```

`rmmod` и `insmod` сейчас не используются. Вместо них – `modprobe`. Он делает это "умно", учитывая зависимости и тд.

Добавить модуль:
```bash
modprobe e1000
```
