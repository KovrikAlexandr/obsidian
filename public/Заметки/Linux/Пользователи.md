## Данные о пользователях

Данные о пользователях хранятся в `/etc/passwd`. Одна строка – один пользователь.

Строка имеет следующую структуру:
```
user:passwd:uid:guid:gecos:home:shell
```

**User**
Имя пользователя

**Passwd**
Раньше здесь хранился хэш пароля. 
Сейчас возможны такие значения:

- `x` – пароль находится в `/etc/shadow`
- `*` или `!` – аккаунт заблокирован

**UID**
ID пользователя. Значения:
- `0` – root
- `1-999` – системные аккаунты
- `1000+` – обычные аккаунты

**GUID**
ID основной группы пользователя (primary group).

**GECOS**
Строковое поле с информацией о пользователе. Может содержать:
- ФИО, описание
- Телефон/Почту

Формат строго не стандартизирован.

**Home**
Домашняя директория пользователя (хомяк). При логине пользователь оказывает в ней

**Shell**
Оболочка для интерактивного входа. Обычно:
- `/bin/bash` или `/bin/zsh`

Можно запретить интерактивный вход, указав:
- `/bin/false`
- `/sbin/nologin`

## Хранение паролей

Пароли и информация и них хранятся в `/etc/shadow`. 
Каждая строка – пароль одного пользователя + дополнительные данные:
```
username:password:last_changed:min_age:max_age:warn:inactive:expire:reserved
```

**Username**
Имя пользователя

**Password**
Захешированный пароль. Пишется в следующем формате:
```
<id>$<salt>$<hash>
```
- `<salt>` – соль
- `<hash>` – хеш
- `<id>` – идентификатор хеш-функции (обычно SHA512)
`*` или `!`означает, что у пользователя нет пароля.

**Last Changed**
Дата последнего изменения пароля. 
Число: количество дней с 1970-01-01 (Unix epoch, но в днях).

**Min age**
Минимальное количество дней, которое должно пройти перед новой сменой пароля. (обычно 0)

**Max age**
Через сколько дней пароль становится просроченным и требуется смена. Обычно 99999 – то есть примерно никогда.

**Warn**
За сколько дней до **max_age** начать предупреждать пользователя сменить пароль

**Inactive**
Сколько дней после истечения пароля аккаунт терпит, прежде чем будет заблокирован

**Expire**
Количество дней, после которого аккаунт блокируется (независимо от **max_age**)

**Reserved**
Не используется
