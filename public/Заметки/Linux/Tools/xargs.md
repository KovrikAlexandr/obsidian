Утилита позволяет принять вывод одной команды и подставить его в вывод другой команды.

Пример:
```bash
printf "a\tb\nc\0c\n" | xargs echo
```

Здесь xargs бьёт вывод на аргументы по разделителям. Разделителями являются пробелы, символы табуляции и перехода строк.

Включать спецсиволы в вывод. Разбивать по `\0`:
```bash
printf "a\tb c\nd\0d\n" | xargs -0 echo
```

Размер батча, который надо передать в один вызов:
```bash
printf "a b c d e f" | xargs -t -n 3 echo
```

Шаблонная подстановка:
```bash
sudo awk -F':' '$1 == "root" {print $3 * 86400}' /etc/shadow | xargs -I{} \
date -d "@{}" +%Y-%m-%d
```

Не запускать, если выход пустой:
```bash
grep -l something *.txt | xargs -r rm
```

Полезные флаги:
- `-r` – не запускать команду, если выход пустой
- `-t` – печатать команду перед вызовом
- `-p` – подтверждение перед каждым запуском
