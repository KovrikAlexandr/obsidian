Ветка – именованный указатель на коммит.
## Виды веток

В git есть три вида веток:

- Локальные ветки (Local branch) – объект на локальной машине. Можно изменять локально

- Удалённые ветки (Remote branch) – ветка в удалённом репозитории

- Отслеживающие ? ветки (Remote-tracking branch) – локальный объект, который отслеживает состояние ветки в удалённом репозитории. Меняется только при взаимодействии с этим репозиторием (`pull`, `fetch`, `push`)

## Операции с ветками

Переключиться:
```bash
git switch dest-branch
```

Создать и переключиться:
```bash
git switch -c new-branch
```
